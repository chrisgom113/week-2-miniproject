Flexbox row

Use these three properties to create a Flexbox row layout. 

                    
                        .row {
                            display: flex;
                            flex-direction: row;
                            flex-wrap: wrap;
                        }

Flexbox Column

Use this to create Flexbox Column layout. 

                    
                        .column {
                            display: flex;
                            flex-direction: column;
                        }

CSS Grid Layout

Build a 12-column layout using CSS Grid 

                    
                        .grid {
                            display: grid;
                            width: 100%;
                            grid-template-columns:
                            repeat(12, 1fr);
                        }
